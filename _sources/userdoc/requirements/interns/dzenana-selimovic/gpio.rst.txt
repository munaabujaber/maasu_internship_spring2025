GPIO Requirements
===========================

.. req:: GPIO pin initialization
   :id: REQ_INT3_1745295115
   :status: Draft

   The system shall support initialization of each GPIO pin based on user configuration as either input or output.

   :submitted-by: Dzenana Selimovic
